理解GCC/G++编译器

一、 编译流程的四个阶段

GCC/G++ 的编译过程分为四个阶段：

1. 预处理

   - 处理宏定义（#defile）、头文件包含（#include）、条件编译（#ifdef）。

   - 使用 -E 选项查看预处理结果：

     ```bash
     g++ -E main.cpp -o main.ii
     ```

2. 编译

   - 将预处理后的代码转换未汇编代码（.s）。

   - 使用 -S 选项生成汇编代码：

     ```bash
     g++ -S main.ii -o main.s
     ```

3. 汇编

   - 将汇编代码转换成机器码（.o 目标文件）。

   - 使用 -c 选项生成目标文件：

     ```bash
     g++ -c main.s -o main.o
     ```

4. 链接

   - 将多个目标文件和库链接为可执行文件。

   - 使用-l指定链接库，-L指定库路径。

     ```bash
     g++ main.o -o a.out
     ```
